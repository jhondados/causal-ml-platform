# ⚗️ Causal ML Platform

[![DoWhy](https://img.shields.io/badge/Powered%20by-DoWhy%20%2B%20EconML-blue)](.) [![Uplift](https://img.shields.io/badge/Uplift%20Model-AUC%200.81-green)](.) [![ROI](https://img.shields.io/badge/Marketing%20ROI-%2B43%25-orange)](.)

> **Causal ML** moves beyond correlation to causation. Uses DoWhy + EconML to estimate true treatment effects, build uplift models and answer "what would have happened?" counterfactuals. **+43% marketing ROI** through precision targeting.

## 🎯 Causal Questions Answered
- "Did the discount actually cause more purchases, or did buyers already intend to buy?"
- "Which customers benefit MOST from a promotional offer?"
- "What is the true ROI of our loyalty program?"
- "Would retention have improved if we had sent the email earlier?"

## 🏗️ Methods Implemented
- **DML (Double ML)**: unbiased treatment effect with nuisance removal
- **X-Learner**: heterogeneous treatment effects for small treated groups
- **CausalForest**: non-parametric CATE estimation
- **Instrumental Variables**: handle unmeasured confounders
